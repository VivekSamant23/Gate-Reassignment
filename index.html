This repository contains a React + Flask application. GitHub Pages can only host static files, so this page is a simple landing page.

Source code

View the repository on GitHub

Run locally

Backend (Flask)

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py

Frontend (Vite)

cd frontend
npm install
npm run dev

Then open: http://localhost:5173

Note

If you want a real web deployment, use a hosting platform that supports a backend (e.g. Render/Fly.io/Heroku) and a frontend (e.g. Netlify/Vercel) or build the frontend and serve it from Flask.
